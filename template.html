メールテンプレートの設定

使用可能な変数:
{{name}} 参加者氏名
{{inviter}} 招待者氏名（※名簿の「招待者」列から自動挿入）
{{date}} 開催日（例: 2026年3月18日）
{{visitorlist}} ビジターリストPDFのURL
{{memberbook}} メンバーブックPDFのURL
共通 Cc (任意・カンマ区切りで複数可)
共通 Bcc (任意・カンマ区切りで複数可)
代理送信 Web App URL (任意)
管理者のGmailアカウントから一括送信するための、別途デプロイした Web App の URL を貼り付けます（デプロイ時に「実行ユーザー: 自分」「アクセス権: 全員」で公開してください）。
空欄の場合は、メニューを開いた人自身のGmailから送信されます。
代理送信用 シークレットトークン
上記Web Appとこのスプレッドシートの両方に同じ文字列を設定してください（推測されにくい任意の文字列で構いません）。
